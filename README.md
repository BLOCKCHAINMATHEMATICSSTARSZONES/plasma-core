# Plasma Library

### Installation

```bash
# using npm
npm install --save @maticnetwork/plasma

# using yarn
yarn add @maticnetwork/plasma
```

### Setup

This monorepo uses Lerna to manage multiple packages.

### Installation

```bash
yarn
```

**Build**

```bash
# this will build all packages
yarn build
```

### Publish

```bash
# change package versions
lerna version x.x.x

# publish packages to npm
lerna publish
```
